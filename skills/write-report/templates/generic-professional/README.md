# Generic Professional Report Template

Use this template for internal reports, technical reports, lab reports, and project summaries when no publisher or venue template is required.

Rules:

- Replace all instructional text before delivery.
- Keep claims traceable to supplied evidence or cited sources.
- Cite every figure and table in the text.
- Use `booktabs` tables and readable figure labels.
- Move long details to the appendix.
- Compile from `main.tex`.
